Etiquetas semanticas estructurales

Es una etiqueta contenedora que no tiene semántica
Cabecera de un sitio web o de una sección
Define la seccion principal del documento, solo puede existir una etiqueta main al igual que el h1 en un documento
Pie de página de un sitio web o de una sección del sitio
Representa una navegación, un menú de enlaces
Representa una seccion autocontenido(que por si solo se explica)
Representa un contenido complementario o secundario. (UNA LISTA DE ARTICULOS RELACIONADOS)
Una seccion de contenido genérico
Representa una direccion física, además una información de contacto

Etiquetas en linea VS Etiquetas en bloque

Etiqueta de bloque por excelencia es el DIV, una etiqueta de bloque es aquella que genera saltos de linea con sus etiquetas hermanas(h1,h2,...). Ocupan todo el ancho del viewport
La etiqueta de linea por excelencia es el SPAN. Una etiqueta de linea solo ocupa el espacio necesario que tiene su contenido Hola soy otra span
